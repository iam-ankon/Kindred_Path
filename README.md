# Kindred Path — Disability Support Services (React + Vite)

A static marketing site inspired by the structure of afford.com.au, rebuilt from
scratch with original branding, copy and illustrations as a React + Vite project.

## Sections
- Sticky header with phone bar + nav
- Hero ("Good friends. Great adventures.")
- Impact stats
- "Who we support" audience picker
- Services (accommodation, community, employment, in-home & coordination)
- Client stories
- FAQ accordion
- Contact + newsletter signup
- Footer with Acknowledgement of Country

## Getting started

```bash
npm install
npm run dev       # start local dev server
npm run build     # production build to /dist
npm run preview   # preview the production build
```

## Stack
- React 18
- Vite 5
- Plain CSS with a design-token system (no UI framework)
- Google Fonts: Baloo 2 (display) + Work Sans (body)
